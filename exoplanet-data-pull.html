Explore NASA Exoplanet Data#

J. Colliander
2022-06-08 (at 3rd Jack Eddy Symposium).

The NASA Exoplanet Institute at Caltech hosts the NASA Exoplanet Archive. This resource provides the data on known exoplanets and mobilizes this data with an applied programming interface (API). The API Queries page provides example calls.

The goals for this short notebook:

  • ingest NASA Exoplanet data via API

  • transform data into Pandas Dataframe

  • start exploration of data
